This program is developed in React.js, Node.js and TypeScript.
Styled with Cascading Style Sheets (CSS). NO NEED FOR SCSS.
Bundled with Babel and Webpack. devDependencies are available in package.json.

To run this project you need Node.js version 14+ (Node.js 16.8.0 is preferd).
Yarn version v1.22.10

After cloning the project you can run the project by following steps:

1-Please direct your command prompt/terminal to the root folder.
(please make sure your command prompt/terminal is directed at the root folder of this project.)

2-To install the dependencies Run:
+ (yarn install) 

@@ To run in production mode @@
3-First we have to build the project using:
+ (yarn build) command.
3.1-Then after build is finished you can start the local server using 
+ (yarn start).

@@ To run in development mode @@
3-To build the development bundle please run:
+ (yarn watch)

3.1-Then after build is finished you can start the local server using 
+ (yarn dev)

@@ To run unit testing @@
3-Pleae run:
+ (yarn test)
